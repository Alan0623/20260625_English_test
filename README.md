# 邏輯測驗 線上練習

從《邏輯測驗》與《英文基礎能力》PDF 題庫整理而成的互動測驗網頁,共 786 題,依題型分為 8 個科目(含英文),並附科目教學分頁與逐題詳解。

## 使用方式
直接用瀏覽器開啟 `index.html`(單一檔案、離線可用),或將整個資料夾上傳至靜態網站主機(GitHub Pages、Netlify、Cloudflare Pages 等)即可部署。`index.html` 不依賴 `data/`,可單獨部署。

左側主選單:測驗練習、錯題練習、答案解析、簡章(可收合、手機版為抽屜)。

## 錯題練習
在「測驗練習」中答錯的題目,會自動彙整到「錯題練習」(側欄顯示數量徽章);在錯題練習中答對後該題即移除。錯題清單以瀏覽器 localStorage 記憶,關閉後再開啟仍保留。
- **測驗練習**:選科目 → 每 10 題一組 → 提交即時批改、計分。
- **成語教學**:前 57 題涉及的 42 則成語,含釋義、出處,可搜尋。

## 重建流程(data/)
- `parse.py` 解析 `raw.txt` → `questions.json`;`parse_eng.py` 解析英文 → `eng_questions.json`。
- 詳解資料:`exp_*.json`(成語由 build 內表生成、數列 `seq_exp.json`、`exp_ciyi/leibi/luoji/yuedu/jisuan.json`、英文 `eng_exp.json`、英文翻譯 `eng_trans.json`)。
- `build_html.py` 套用校正、分類、詳解、翻譯、表格還原、成語表與單字卡,輸出 `index.html`。

## 部署
`index.html` 為自包含單檔(所有題庫與資料內嵌),直接放到任何靜態主機根目錄即可;`data/` 為原始檔與重建腳本,部署時可不上傳。

## 重建流程(原始檔與腳本/)
1. `parse.py` — 從 `raw.txt`(PDF 抽取文字)解析出 `questions.json`(題號、科目、題幹、選項、答案)。
2. `build_html.py` — 套用校正與分類,合併成語教學資料,輸出 `邏輯測驗.html`。
3. `原始題庫.pdf` — 原始來源檔。

修改題目或分類後,重跑 `parse.py` 再跑 `build_html.py` 即可重新產生網頁。
